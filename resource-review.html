R for Data Science - Hadley Wickham

R for Data Science (R4DS) is available for free and is the definitive guide to using R to conduct data science. I've purchased a hard copy because I found the content in R4DS to be absolutely invaluable. R4DS takes you from knowing nothing about R to creating predictive models in 522 enjoyable, informative, and engaging pages. If you can only read one about R, make this the one.

Verdict - Buy It



Hands on Programming with R - Garrett Grolemund

Garret Grolemund co-authored R4DS, so it's no wonder that Hands on Programming with R is top-notch. The book assumes no prior knowledge in R and walks you through three interesting projects. The projects focus on programming in R, teaching you about environments, the object oriented S3 system, creating simulations, writing vectorized code, and a few other topics all in an applied setting. Definitely have a look at this book.

Verdict - Read It



Statistics Using R - Sudha Purohit

Printed in 2008, Statistics Using R features muddled instructions, poor formatting, out of date programming techniques, and half-explained theory. If I wasn't taking a probability course, I doubt I would be able to follow along the chaotic and assuming narrative of Statistics Using R. Don't waste your valuable time with this one.

Verdict - Skip It



Learn Python the Hard Way - Zed Shaw

Widely acclaimed, Learn Python the Hard Way provides a "from the beginning" introduction to general programming in python. Zed emphasized rote memorization and learning by repeatedly doing through a collection of 53 exercises and accompanying 'study drills'. There is an exponential curve in difficulty, as the first ~35 chapters are a bit boring and repetitive, while the last ~15 chapters are quite difficult and ill-explained. Personally, I would have preferred a more linear progression in difficulty, and don't care too much for the 'drill' style of learning.

Verdict - Skip It



Think Python - Allen Downey

I really enjoyed working through Think Python. The 240-page text provides a compact introduction to general programming with Python. Note that there is no discussion of data science packages, the book is simply an introduction to the language and foundational concepts. With great prose, and a good mix of example code and explanation, Think Python is a must read for those without prior experience in Python. The second edition was released in 2015, so it is written in Python 3 and is still highly relevant.

Verdict - Read It
